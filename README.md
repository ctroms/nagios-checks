### Nagios Plugins

Collection of Nagios plugins

check_core_dumps
-----------

uses $coredir to scan for core dumps, alerts to nagios.  Requires Nagios/Plugin/Functions.pm.
    perl check_core_dumps
    OK: Total Coredumps: 0
